  • AutoMockr

    AutoMockr is a Spring Boot-based utility designed for intelligent random data generation using custom annotations. It enhances JPA repository operations by automatically populating missing or null fields with context-aware mock data during persistence—without requiring any additional method calls or boilerplate logic.

    Key features include:

    • Seamless integration with standard save() methods from JpaRepository.
    • Three custom annotations that auto-generate relevant data for entity fields if left unspecified.
    • Smart contextual generation—for instance, generating an email and address based on a given name.
    • Supports composite models like Customer (with Address as a nested object), and stores data in a MySQL database.

    This project demonstrates advanced Java skills, Spring Boot expertise, and practical use of annotations, reflection, and data modeling.
